Le module Philips TV est compatible avec certains téléviseurs Philips utilisant l'API JointSpace. Il faudra fournir l'adresse IP du téléviseur (pour la trouver : touche HOME, Configuration, Installation, Réseau, Voir paramètres réseau, " adresse IP ")

Ordres:

Le module permet de piloter le téléviseur avec les fonctions classiques:

  • Sarah met la chaine une
  • Sarah met Canal Plus
  • Sarah monte le volume
  • Sarah coupe le son
  • ...

Macro:

La commande WAIT permet de créer une pause d'une seconde. Cela peut être utile pour attendre le chargement du menu par le téléviseur (Voir exemple Macro dans le fichier XML).

met la télé sur la freebox  out.action.commande="Source,WAIT,CursorDown,WAIT,Confirm";
