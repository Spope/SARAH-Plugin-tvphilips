# [S.A.R.A.H.](http://encausse.net/s-a-r-a-h)

This plugin is an add-on for the framework [S.A.R.A.H.](http://encausse.net/s-a-r-a-h), an Home Automation project built 
on top of:
* C# (Kinect) client for Voice, Gesture, Face, QRCode recognition. 
* NodeJS (ExpressJS) server for Internet of Things communication


## Description

This plugin is a connector for some Philips TV using [JointSpace API](http://jointspace.sourceforge.net/).


## Quick Start

Donwload the plugin from SARAH's AppStore and follow online plugin's documentation
